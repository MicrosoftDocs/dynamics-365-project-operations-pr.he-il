# <a name="create-product-catalog-items-project-service"></a>יצירת פריטים של קטלוג מוצרים (Project Service)

 לקבלת מידע נוסף על הוספת מוצרים לקטלוג המוצרים, ראה [יצירת מוצר (Sales)](https://docs.microsoft.com/dynamics365/sales-enterprise/create-product-sales).
