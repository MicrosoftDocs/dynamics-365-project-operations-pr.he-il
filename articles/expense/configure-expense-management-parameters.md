# <a name="configure-expense-management-parameters"></a>קביעת התצורה של פרמטרי ניהול הוצאות

_**חל על:** ‏Project Operations לתרחישים מבוססי משאבים/ללא מלאי_

נושא זה מתאר את הפרמטרים השולטים באופן הפעולה הכללי בניהול הוצאות.

## <a name="general"></a>כללי

| שדה                                                    | תיאור |
|----------------------------------------------------------|-------------|
| תעריף רגיל עבור מרחק                                 | הזן את שיעור ההחזר עבור הוצאות מרחק. שיעור זה מוכפל במרחק שהוזן עבור ההוצאה לצורך חישוב הסכום המוחזר עבור הוצאות הנסיעה. |
| אימות מטרת ההוצאות                                 | הפעל אפשרות זו כדי להגביל את המשתמשים לקבוצת ערכים קיימת המוגדרת בשדה **מטרת דוח ההוצאות** כאשר הם יוצרים דוחות של הוצאות. |
| הוצאות אישיות משולמות על ידי                                | בחר באדם האחראי לתשלום סכומי עסקאות בכרטיס אשראי המסווגים כאישיים. |
| הצגת דוח הוצאות שלם להסתעפות לאחור               | בחר באפשרות זו כדי להציג את כל ההוצאות עבור דוח הוצאות כאשר מוצגים פרטי המסמך המקורי עבור שובר ספציפי שנוצר בעת רישום דוח ההוצאות. |
| מתן הרשאה מראש לנסיעה הוא חובה                 | בחר באפשרות זו כדי לדרוש כי דרישת נסיעה תוגש ותאושר לפני שעובד יכול להגיש דוח הוצאות. |
| אפשר עריכה של הפצות לפני הרישום            | בחר האם ניתן יהיה לערוך את ההפצות של דוח הוצאות לפני רישומו. |
| הערך את מדיניות ניהול ההוצאות                     | בחר מתי יש לבצע הערכה להוצאות כדי לקבוע אם מדיניות הוצאות הופרה. ניתן להעריך הוצאות בגין הפרות כאשר ערך ההוצאה הוזנה ונשמרה, או כאשר ההוצאה מוגשת לאישור. כללי המדיניות לקבלות הנדרשות יוערכו לאחר שמירת שורת ההוצאות. |
| אפשר שורות הוצאות בין-חברתיות                         | בחר אם ניתן להזין הוצאות עבור ישויות משפטיות אחרות בדוח הוצאות. |
| אפשר עריכת שער החליפין של הוצאות כרטיסי אשראי | בחר באפשרות זו כדי לאפשר למשתמש לערוך את שער החליפין עבור הוצאות בכרטיסי אשראי שיובאו. |
| שדות היררכיה מרובי רמות להצגה                  | בחר אלו שדות של מאשר יוצגו כאשר סוג הקצאת זרימת העבודה מוגדר כהיררכיה, והבחירה **היררכיה** מוגדרת להשתמש בהיררכיית האישור, אישור בעל רמות מרובות להוצאות. כאשר נעשה שימוש בהיררכיית אישור מרובת רמות עבור זרימת עבודה, השדות שנבחרו יוצגו בדוח ההוצאות וניתן יהיה לערוך אותם. |
| הזן מספר כרטיס אשראי של העובד                        | בחר אם ניתן יהיה להזין ולשמור מספר בן 15 תווים או 16 תווים בשדה **מזהה כרטיס** בדף **כרטיסי אשראי** של עובד. |
| אמת את מטרת דרישת הנסיעה                      | יש להפעיל אפשרות זו כדי להגביל את המשתמשים לקבוצת ערכים קיימת המוגדרת בשדה **מטרת דוח ההוצאות** כאשר הם יוצרים דרישות נסיעה. |

## <a name="financial"></a>בנקאות ושירותים פיננסיים

| שדה                                                                                    | תיאור |
|------------------------------------------------------------------------------------------|-------------|
| שם היומן היומי של הספר הראשי                                                                | בחר את שם יומן הספר הראשי שבו נרשמים דוחות הוצאות שאושרו. |
| אפשר החזר מס מהוצאות                                                        | בחר באפשרות זו כדי לאפשר החזר מס על הוצאות בגין הוצאות זכאיות. לא ניתן לבחור באפשרות זו אם מופעלים כללי מס המכירות והשימוש של ארה"ב. |
| רישום מקדמות במזומן באופן מיידי                                                           | בחר באפשרות זו כדי לרשום מקדמה במזומן שאושרה לאחר השלמת תהליך התשלום וההעברה. אם אפשרות זו לא נבחרה, תהליך התשלום וההעברה יוצר יומן כללי שלא נרשם. |
| תאריך חשבונאות נכון במהלך הרישום                                                   | בחר באפשרות זו כדי לעדכן את תאריך החשבונאות שבו נרשמות הוצאות, אם התקופה הנוכחית בהמתנה או סגורה. תאריך החשבונאות ייקבע לתקופה הפתוחה הבאה. |
| אפשר קיבוץ עסקאות על בסיס חשבון קיזוז שצוין בשיטת התשלום       | בחר באפשרות זו כדי לסכם את עסקאות ההוצאות עבור דוח הוצאות, בהתבסס על חשבון הקיזוז שצוין בשיטת התשלום עבור ההוצאות. |
| מס כלול                                                                             | בחר באפשרות זו כדי לכלול מס מכירה בסכום ההוצאות כברירת מחדל. |
| שחרר התחייבויות בסגירת דרישות נסיעה                                      | בחר באפשרות זו כדי לשחרר כספים משועבדים בעת סגירת דרישת הנסיעות שאושרה. |
| אפשר הגשת דרישת נסיעות בחריגה מהתקציב לפנקס התקציב ותקציב הפרויקט | בחר באפשרות זו כדי לאפשר לעובדים להגיש בקשות נסיעה לאישור, למרות שהם חורגים מהתקציב המותר שנקבע בפנקס התקציבים או מהתקציב שהותר לפרויקט. |
| אפשר הגשת דרישת נסיעות בחריגה מהתקציב עבור פנקס התקציבים ותקציב הפרויקט     | בחר באפשרות זו כדי לאפשר לעובדים להגיש דוחות הוצאות לאישור, למרות שהם חורגים מהתקציב המותר שנקבע בפנקס התקציבים או מהתקציב שהותר לפרויקט. |
| אפשר אישור דוח הוצאות בחריגה מהתקציב עבור פנקס התקציבים בלבד                 | בחר באפשרות זו כדי לאפשר למאשרים לאשר דוחות הוצאות החורגים מהתקציב המותר שנקבע בפנקס התקציבים. |

## <a name="per-diem"></a>אש"ל

| שדה                                 | תיאור |
|---------------------------------------|-------------|
| שעות מינימום לאש"ל            | הזן את מספר השעות המינימלי המוגדר כברירת מחדל שעובד חייב לעבוד ביום כדי להיות זכאי לקבלת דמי אש"ל עבור הוצאות הקשורות לנסיעה. ערך זה משמש כערך ברירת מחדל רק עבור השדה **שעות מינימום** לרמות תעריפי אש"ל. |
| אחוז ארוחות                          | הזן את אחוז ברירת המחדל של דמי האש"ל עבור ארוחות, המשמש ליום הראשון והאחרון של הוצאה הקשורה לנסיעה, כאשר השדה **חישוב הפחתת הארוחה לפי** מוגדר ל **סוג הארוחה ליום** אוֹ ל **מספר הארוחות ביום**. יום העבודה ביום הראשון והאחרון עשוי להיות קצר יותר מיום העבודה הרגיל. לפיכך, סכום האש"ל בימים אלו עשוי להיות שונה מהסכום הרגיל. אם האחוז מוגדר ל-**0** (אפס), הניכויים ליום הראשון והאחרון יהיו 0.00. |
| אחוז המלונות                         | הזן את אחוז ברירת המחדל של האש"ל המשמש למלונות ביום הראשון והאחרון מההוצאה הקשורה לנסיעה. יום העבודה ביום הראשון והאחרון עשוי להיות קצר יותר מיום העבודה הרגיל. לפיכך, סכום האש"ל בימים אלו עשוי להיות שונה מהסכום הרגיל. אם האחוז מוגדר ל-**0** (אפס), הניכויים ליום הראשון והאחרון יהיו 0.00. |
| אחוזים אחרים                         | הזן את אחוז ברירת המחדל של האש"ל המשמש להוצאות שונות ביום הראשון והאחרון מההוצאה הקשורה לנסיעה. יום העבודה ביום הראשון והאחרון עשוי להיות קצר יותר מיום העבודה הרגיל. לפיכך, סכום האש"ל בימים אלו עשוי להיות שונה מהסכום הרגיל. אם האחוז מוגדר ל-**0** (אפס), הניכויים ליום הראשון והאחרון יהיו 0.00. |
| הפחתת אחוזים לארוחת הבוקר | הזן את הסכום הפחתה מהאש"ל לארוחת הבוקר. לדוגמא, אם עובד מקבל ארוחת בוקר בחינם, יתכן שתרצה להפחית את סכום האש"ל ב-10 אחוזים. |
| הפחתת אחוזים לארוחת צהריים     | הזן את הסכום הפחתה מהאש"ל לארוחת צהריים. לדוגמא, אם עובד מקבל ארוחת צהריים בחינם, יתכן שתרצה להפחית את סכום האש"ל ב-15 אחוזים. |
| הפחתת אחוזים לארוחת ערב    | הזן את הסכום הפחתה מהאש"ל לארוחת ערב. לדוגמא, אם עובד מקבל ארוחת ערב בחינם, יתכן שתרצה להפחית את סכום האש"ל ב-25 אחוזים. |
| חישוב הפחתת הארוחה לפי           | ציין כיצד על המערכת לחשב את ההפחה מהארוחות באש"ל, על ידי בחירה אם ההפחתה תתבסס על סוג הארוחה לנסיעה או ליום, או על מספר הארוחות המותרות ליום. |
| עיגול דמי אש"ל                     | בחר את סוג העיגול המשמש להוצאות אש"ל. אם בוחרים בעיגול רגיל, כל הוצאת אש"ל עם סכום של 0.50 מעוגלת ל-1.00, וכל הוצאה לכל יום עם סכום הנמוך מ-0.50 מעוגלת כלפי מטה ל-0.00. |
| בסס את חישוב האש"ל על          | בחר אם סכום אש"ל מבוסס על יום קלנדרי או על תקופה של 24 שעות. |

## <a name="fax-cover-pages"></a>עמודי שער של פקס

| שדה                          | תיאור |
|--------------------------------|-------------|
| הוראות                   | הזן את ההוראות שעל העובדים לבצע כאשר הם יוצרים עמוד שער של פקס המשמש למשלוח קבלות שקשורות לדוח הוצאות. להזנת טקסט ספציפי לשפה שיוצג בהתבסס על שפת המשתמש, בחר **תרגומים**. |
| מזהה משתמש (פרטי ברקוד) | בחר באפשרות זו כדי לאחסן את המזהה הייחודי של העובד בברקוד המופיע בעמוד השער של הפקס. |
| ברקוד                       | בחר את הברקוד המשמש בעמוד השער של הפקס. ברקודים 39 ו-128 נתמכים בניהול הוצאות. |

## <a name="anti-corruption"></a>נגד שחיתות

| שדה                                 | תיאור |
|---------------------------------------|-------------|
| הצגת ההצהרה נגד שחיתות   | בחר באפשרות זו כדי להציג את הטקסט נגד שחיתות בעת יצירת דוח הוצאות. לאחר מכן ניתן להפוך לזמינות קטגוריות ספציפיות של הוצאות, שידרשו לבחור את ההצהרה נגד שחיתות בדוח ההוצאות. לדוגמא, קטגוריית מתנות שקשורה להוצאות של פקיד ממשלתי עשויה לדרוש שהעובד יאשר כי ההוצאה עומדת במדיניות החברה שקשורה לפקידי ממשלתיים. |
| הודעה נגד שחיתות למגיש | הזן את הטקסט שיש להציג לעובד שיוצר דוח הוצאות. להזנת טקסט ספציפי לשפה שיוצג בהתבסס על שפת המשתמש, בחר **תרגומים**. |
| הודעה נגד שחיתות למאשר  | הזן את הטקסט שיש להציג למאשר כאשר נוצרת דוח הוצאות. להזנת טקסט ספציפי לשפה שיוצג בהתבסס על שפת המשתמש, בחר **תרגומים**. |
